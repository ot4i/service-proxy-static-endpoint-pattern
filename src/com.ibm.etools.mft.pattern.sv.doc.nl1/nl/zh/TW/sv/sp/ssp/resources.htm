Service Proxy: static endpoint 型樣的範例資源

Service Proxy: static endpoint 型樣提供一組範例資源。範例資源包括訊息集及可部署的 WSDL。範例資源可協助您開始使用此型樣。

附註：只有在使用已整合 IBM Integration Toolkit 的資訊中心時，才可以使用資訊中心中的鏈結來匯入和移除範例資源。

您可以按下列鏈結，來設定範例資源：

  • 將範例資源匯入至工作區

儲存 OrderService.wsdl 來取得原始 WSDL 檔案，以使用用戶端應用程式進行測試。

當您完成範例資源時，可以按下列鏈結，來移除它們：

  • 移除工作區中的範例資源
