Service Proxy: static endpoint 模式的参数

服务信息
模式参数 M C 缺省值 描述
服务提供者的 WSDL 是 否  

此参数定义可部署的 WSDL 文件，该文件定义为其创建代理的服务的接口。

必须从工作空间消息集中选择可部署的 WSDL 文件。
服务提供者的 URL 否 是  

此参数定义正在代理的服务所在的端点。

如果未指定，那么将通过 WSDL 文件中的绑定信息确定端点。
代理 URL 否 是  

此参数定义用于定义已部署代理端点的 URL 路径。

完整的服务地址还包含主机名和端口，这些值将在部署代理时确定。

如果未指定任何路径，那么将使用服务 WSDL 中的路径。
SOAP 请求的验证 否 是 无

根据此参数的定义验证入局 SOAP 消息。

有效值为：

  • 无
  • 内容和值
SOAP 响应的验证 否 是 无

根据此参数的定义验证 SOAP 响应。

有效值为：

  • 无
  • 内容和值
记录
模式参数 M C 缺省值 描述
需要记录 否 否 取消选中

此模式参数确定模式实例是否包括用于记录的代码。

如果已取消选中需要记录，那么将不包括日志子流。

如果已选择需要记录，那么消息流中将包括日志子流，并且必须设置日志队列管理器和日志队列的值。
日志队列管理器 否 是

此模式参数定义用于记录的队列管理器。 如果对记录使用代理队列管理器，那么此参数可留为空白。

仅当选择了需要记录时才需要此参数。

日志队列 否 是 LOG

此模式参数定义用于记录的队列。

仅当选择了需要记录时才需要此参数。
错误处理
模式参数 M C 缺省值 描述
需要错误消息 否 否 已选中

此模式参数定义模式应用程序是否创建消息流元素，该元素创建错误消息。

如果已选择需要错误消息，那么必须设置错误队列管理器和错误队列的值。
错误队列管理器 否 是  

此模式参数定义错误消息的队列管理器。 如果对记录使用代理队列管理器，那么此参数可留为空白。

错误队列 否 是 ERROR

此模式参数定义错误消息的队列。

仅当选择了需要错误消息时才需要此参数。

常规
模式参数 M C 缺省值 描述
代理模式 否 否 mqsi

如果定义了代理模式，那么将在该模式中创建消息流和 ESQL 文件。 如果未定义模式，那么将使用缺省代理模式。
流前缀 否 否  

如果设置了此参数，那么会将此前缀添加到所有消息流名称的开始处。

您仅可以使用在消息流名称中有效的字符。
流后缀 否 否  

如果设置了此参数，那么会将此后缀添加到所有消息流名称的结尾处。

您仅可以使用在消息流名称中有效的字符。
队列前缀 否 否  

如果设置了此参数，那么会将此前缀添加到所有队列名称的开始处。

如果使用了缺省队列名称，那么必须设置前缀和/或后缀，以标识每个模式实例的队列名称。

队列后缀 否 否  

如果设置了此参数，那么会将此后缀添加到所有队列名称的结尾处。

如果使用了缺省队列名称，那么必须设置前缀和/或后缀，以标识每个模式实例的队列名称。

简短描述 否 否  

此描述将被添加到生成的所有消息流的简短描述属性。
详细描述 否 否  

此描述将被添加到生成的所有消息流的详细描述属性。

列出所有模式参数及其缺省值。

标题为 M 的列指示属性是否为必需的。 标题为 C 的列指示属性是否为可配置的，也就是说，可以在将消息流添加到代理归档 (BAR) 文件时更改值。

未表明缺省值时，缺省值是空字符串。
