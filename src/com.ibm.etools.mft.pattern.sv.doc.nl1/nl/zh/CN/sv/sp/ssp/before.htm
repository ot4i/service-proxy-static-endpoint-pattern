应用 Service Proxy: static endpoint 模式之前要完成的任务

应用模式以生成模式实例之前，您必须确定需要执行以下哪些任务。如果需要，请完成以下任务：

如果需要，请完成以下必需任务：

  • 确定此模式是否符合业务需求，以及请求者应用程序和提供者应用程序是否都支持此模式。
  • 定义目标服务接口。
  • 对模式实例进行命名。识别模式应用程序的用途，以便确定合适的名称。
  • 确定是否需要记录。如果需要记录，那么必须确定将记录消息发送到哪个队列管理器和队列。记录必须发送到常规日志队列，其他代理流可能已有该队列。记录可以发送到针对您要创建的模式实例所创建的队列。
  • 确定是否需要错误消息。
  • 确定是否需要验证。
  • 在消息集项目中创建可部署的 WSDL，请参阅 IBM Integration 文档中的“生成 WSDL”。
