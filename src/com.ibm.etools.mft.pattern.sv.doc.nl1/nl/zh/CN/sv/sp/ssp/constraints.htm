使用 Service Proxy: static endpoint 模式时的约束

使用此模式时会有若干约束。

只有在满足以下所有约束时，才可以使用 Service Proxy: static endpoint 模式：

  • 请求的接收形式必须是 HTTP 或 HTTPS 上的 SOAP 消息。
  • 请求应用程序必须支持同步请求响应交互样式。
  • 目标服务必须提供 HTTP 或 HTTPS 上的 SOAP 绑定。
  • 定义目标服务接口的 WSDL 可以具有多个端口类型和绑定，但是只有第一个端口类型或绑定可用于配置生成的模式实例。您可以在消息流编辑器中选择备用项。
  • 如果需要记录，那么会假定此模式使用的格式符合您自己环境的需求。
