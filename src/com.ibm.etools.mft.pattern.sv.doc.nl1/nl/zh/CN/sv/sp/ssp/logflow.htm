Service Proxy: static endpoint 模式的日志子流

启用记录时，日志子流将根据用户定义的属性 RequestLoggingOn 和 ResponseLoggingOn 来记录请求和应答。

日志消息将作为持久 WebSphere MQ 消息写入 WebSphere MQ 队列。将根据您组织中的标准做法来单独处理这些日志消息。消息日志的 MQRFH2 头中包含原始消息和以下信息：

  • 时间戳记
  • 代理名称
  • 消息流名称
  • 服务详细信息
  • 状态（1 = 请求，2 = 应答，-2 = 故障）
