Recursos de Amostra para o Padrão de Service Proxy: Terminal Estático

O padrão de Service Proxy: Terminal Estático fornece um conjunto de recursos de amostra. Os recursos de amostra incluem um conjunto de mensagens e um WSDL implementável. Os recursos de amostra ajudam a iniciar o uso deste padrão.

Nota: É possível usar os links no centro de informações para importar e remover os recursos de amostra apenas ao usar o centro de informações que está integrado com o IBM Integration Toolkit.

É possível configurar os recursos de amostra clicando no seguinte link:

  • Importe os recursos de amostra para sua área de trabalho

Salve o OrderService.wsdl para obter o arquivo WSDL original para testar com aplicativos clientes.

Ao concluir os recursos de amostra, é possível removê-los clicando no seguinte link:

  • Remova os recursos de amostra de sua área de trabalho
