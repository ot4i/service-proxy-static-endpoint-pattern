Virtualização de Serviço

A Virtualização de Serviço é uma das categorias comuns de padrões para soluções de conectividade.

Os padrões de Virtualização de Serviço fornecem loose coupling entre serviços fornecendo níveis adicionais de direção por meio de um Barramento de Serviço Corporativo. Estes padrões também abordam os requisitos de mediação (por exemplo, roteamento, conversão de protocolo, transformação de dados e criação de log) entre serviços ao endereçar requisitos de conectividade em uma Arquitetura Orientada a Serviços. Os padrões de virtualização de serviço geralmente são usados para abordar vários requisitos distintos, incluindo provisão de:
  • Loose coupling apresentando serviços existentes como serviços virtuais no Barramento de Serviço Corporativo.
  • Uma fachada do provedor para incluir transformação e enriquecimento.
  • Uma fachada do provedor para fornecer uma interface padrão para serviços não padrão.
  • Utilitários de gerenciamento de serviço padrão, por exemplo, segurança, criação de log, monitoramento e cobrança.
