서비스 프록시: 정적 엔드포인트 패턴의 사용 제한조건

이 패턴을 사용할 때는 많은 제한조건이 존재합니다.

다음과 같은 제한조건을 모두 충족하는 경우에만 서비스 프록시: 정적 엔드포인트 패턴을 사용할 수 있습니다.

  • 요청이 HTTP 또는 HTTPS를 통한 SOAP 메시지로 수신되어야 합니다.
  • 요청 애플리케이션이 동기화된 요청 응답 상호 작용 스타일을 지원해야 합니다.
  • 대상 서비스가 HTTP 또는 HTTPS를 통한 SOAP 바인딩을 제공해야 합니다.
  • 대상 서비스 인터페이스를 정의하는 WSDL이 여러 개의 포트 유형과 바인딩을 가질 수 있지만, 생성된 패턴 인스턴스 구성에는 첫 번째 포트 유형 또는 바인딩만 사용됩니다. 메시지 플로우 편집기에서 대체 항목을 선택할 수 있습니다.
  • 로그 기록이 필요하면 이 패턴에 사용된 형식이 사용자 고유 환경의 요구사항을 충족한다고 가정합니다.
