이 패턴 매개변수는 패턴 인스턴스가 로그 기록 코드를 포함하는지 여부를 판별합니다.

모든 패턴 매개변수가 기본값과 함께 나열됩니다.

M 열은 특성이 필수인지 여부를 표시합니다. C 열은 특성이 구성 가능한지, 즉 메시지 플로우를 브로커 아카이브(BAR) 파일에 추가할 때 값을 변경할 수 있는지 여부를 표시합니다.
