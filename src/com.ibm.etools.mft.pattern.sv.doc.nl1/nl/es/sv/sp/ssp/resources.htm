Recursos de ejemplo para el patrón Proxy de servicio: punto final estático

El patrón Proxy de servicio: punto final estático proporciona un conjunto de recursos de ejemplo. Los recursos de ejemplo incluyen un conjunto de mensajes y un WSDL desplegable. Los recursos de ejemplo le ayudan a empezar a utilizar este patrón.

Nota: Puede utilizar los enlaces del centro de información para importar y eliminar los recursos de ejemplo sólo si utiliza el centro de información que está integrado en IBM Integration Toolkit.

Puede configurar los recursos de ejemplo pulsando el enlace siguiente:

  • Importar los recursos de ejemplo al espacio de trabajo

Guarde el archivo OrderService.wsdl para obtener el archivo WSDL original para probarlo con aplicaciones cliente.

Cuando haya terminado con los recursos de ejemplo, puede eliminarlos pulsando el enlace siguiente:

  • Eliminar los recursos de ejemplo del espacio de trabajo
Volver a la especificación del patrón Proxy de servicio: punto final estático
