Exemples de ressources pour le modèle Proxy de service : noeud final statique

Le modèle Proxy de service : noeud final statique propose un ensemble d'exemples de ressources. Les exemples de ressources comprennent un ensemble de messages et un WSDL déployable. Les exemples de ressources vous aident à commencer à utiliser ce modèle.

Remarque : Vous pouvez utiliser les liens du centre de documentation pour importer et supprimer les ressources exemple uniquement lorsque vous utilisez le centre de documentation qui est intégré à IBM Integration Toolkit.

Vous pouvez configurer les exemples de ressources en cliquant sur le lien suivant :

  • Importez les exemples de ressources dans votre espace de travail

Enregistrez OrderService.wsdl pour obtenir le fichier WSDL d'origine pour effectuer un test avec les applications client.

Lorsque vous en avez terminé avec les exemples de ressources, vous pouvez les supprimer en cliquant sur le lien suivant :

  • Supprimez les exemples de ressources de votre espace de travail
